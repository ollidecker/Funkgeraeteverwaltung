# Funkgeräteverwaltung Version 1.0.2

Browserbasierte Funkgeräte-Ausgabe ohne Node.js/React.

# Installation Debian:
bash <(curl -s https://raw.githubusercontent.com/ollidecker/Funkgeraeteverwaltung/main/install.sh)

# Installation direkt auf Proxmox als Container:
bash <(curl -s https://raw.githubusercontent.com/ollidecker/Funkgeraeteverwaltung/main/install-proxmox-lxc.sh)

# Update direkt im LXC
curl -s https://raw.githubusercontent.com/ollidecker/Funkgeraeteverwaltung/main/update.sh | bash



## Start Mac
ZIP entpacken und `start-mac.command` doppelklicken.
In Einstellungen evtl. Sicherheitsabfrage überbrücken.
Danach im Browser öffnen:
http://127.0.0.1:14943 bzw. das was in der letzten Zeile des Terminals steht.
